# Manifesto of Coherence
### A Constitution of Eternal Living · v9

> *"To live in coherence is to live in truth. To act in coherence is to act in love."*

---

## What This Is

This is a living constitutional document — a personal governing framework built on a single founding axiom: **the field is real.**

It is not a self-help document. It is not a belief system. It is a structural description of how existence works, and a set of laws derived from that description — written to govern one life, enforced through awareness and action.

The document has been in continuous revision since its first inscription. This repository is its permanent home and its public record.

## Structure

The constitution is organized as a single self-contained HTML file with the following architecture:

**Constitutional Framework**
- Declaration of Authority — establishes what this document is and who it governs
- Article Zero — the founding axiom that precedes all argument

**The Manifesto Body** *(eleven sections)*
- I. The Field

- IV. Motion & Time
- V. Silence
- VI. Coherence
- VII. Growth

- X. The Way

**Constitutional Apparatus**
- Bill of Rights — six inviolable conditions
- Amendment Process — the governed procedure for revision

- The Oath — five lines. Memorizable. The living core.
- Ratification — where the document becomes law

**Record**
- Changelog — revision history from v1 to present

## Technical Features

The document is a single HTML file with no external dependencies beyond two Google Fonts. Everything else is built in.

**Visual**
- Animated toroidal field rendered on a fixed canvas behind the text — 7% opacity, always present, never decorative
- WebGL Milky Way galaxy (145,000 individually positioned stars) that fades in at Article Zero and disappears after *"The field is real."*
- Day mode sun — a living plasma rendering with corona streamers, magnetic loop arcs, chromosphere ring, granulation turbulence, surface sparkle, and 280 corona spark particles
- Gold progress thread tracking reading position along the left edge
- Scroll-reveal on all sections
- Day / night mode toggle — night is gold on obsidian, day is navy on warm parchment with grain texture

**Audio**
- Ambient binaural soundscape generated entirely via Web Audio API — no audio files
- 396Hz drone split binaurally with a 7Hz theta-wave beat (left 396Hz / right 403Hz)
- 528Hz harmonic overtone pad, 198Hz sub-harmonic, slow LFO breathing, 1188Hz shimmer
- Fades in over 6 seconds on first scroll or interaction

**Typography**
- Cinzel for titles, numerals, and labels — Roman stone-carving lineage
- Cormorant Garamond for body — Renaissance manuscript warmth
- Root font size 19px — scaled for sustained reading
